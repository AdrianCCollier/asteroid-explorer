This is a basic react application that shows how to use phaser can be used inside of react

When the application starts index.html will run as well as index.js

from here index.js will run App.js which is what handles all the pages inside of the application, and the switching between pages

the game canvas can be seen when the page is switched from the landing page, in this case the game canvas is Luffy bouncing in a game screen using basic
phaser physics
